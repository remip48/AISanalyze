# Package index

## Functions

Core functions for analysing AIS data.

- [`AIStravel()`](https://remip48.github.io/AISanalyze/reference/AIStravel.md)
  : Calculate vessel travel metrics
- [`AIScorrect_speed()`](https://remip48.github.io/AISanalyze/reference/AIScorrect_speed.md)
  : Correct GPS errors in AIS tracks
- [`AISidentify_stations_aircraft()`](https://remip48.github.io/AISanalyze/reference/AISidentify_stations_aircraft.md)
  : Identify AIS base stations and high-speed craft
- [`AISinterpolate()`](https://remip48.github.io/AISanalyze/reference/AISinterpolate.md)
  : Interpolate AIS positions
- [`AISextract()`](https://remip48.github.io/AISanalyze/reference/AISextract.md)
  : Extract AIS positions around target locations and times
- [`AISinfos()`](https://remip48.github.io/AISanalyze/reference/AISinfos.md)
  : Estimate vessel characteristics
